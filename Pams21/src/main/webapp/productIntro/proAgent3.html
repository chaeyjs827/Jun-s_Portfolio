S2/Lite는 PAMS21(Perfect Agent Management System 21C)을 기본으로 Upgrade된 프로그램으로 소규모 대리점에서 적은 비용으로 Mobile 환경을 구축하여 정확하고 빠른 업무처리를 지원합니다.

S2Lite 연관제품

  • 업무흐름도
  • 주요화면
  • 지원기능
  • 시스템구성도
  • 모바일장비

  • 제품등록

  • 거래처등록

  • 판매등록

  • 거래처원장

  • 미수금현황

  • 거래처별 판매집계표

  • 실재고등록

  • 세금계산서발행

  • 전자세금계산서작업
